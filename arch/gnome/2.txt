===Группы пользователей===
1. Создание пользователя (без прав суперпользователя)
sudo useradd -m -s /bin/bash имя_пользователя
sudo passwd пароль   

2. Создание группы
bash
sudo groupadd название_группы
3. Добавление пользователя в группу
sudo usermod -aG название_группы имя_пользователя
4. Проверка групп пользователя
groups имя_пользователя


===Настройка аутентификации и авторизации===
Проверка наличия второго пользователя и его прав

На экране блокировки выбираем второго пользователя, вводим пароль
В терминале проверяем наличие у него прав суперпользователя
sudo su
Вводим пароль
Должно отобразиться сообщение "имя_пользователя is not in the sudoers file."


===Журнал мониторинга===
Проверка активности журнала мониторинга логов:
systemctl status systemd-journald

Проверка логов текущей загрузки:
journalctl -b

Проверка логов конкретной службы (например, SSH)
journalctl -u sshd
